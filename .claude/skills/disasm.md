# Disassemble goTS Bytecode

Show the disassembled bytecode for a goTS source file.

## Usage
Run this skill to inspect the generated bytecode.

## Command
```bash
cd /Users/yang/workspace/quickts/gots && go run ./cmd/gots disasm test/example.gts
```

## When to use
- Debugging compiler output
- Understanding bytecode generation
- Verifying opcode emission
